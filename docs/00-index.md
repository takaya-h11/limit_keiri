# リミット四ツ谷店 売上管理連携MCPサーバー - チケット管理

## プロジェクト概要
LINEで受信した売上報告をGeminiが解釈し、Googleスプレッドシートに自動入力するMCPサーバー

## チケット一覧

### ✅ [チケット01: プロジェクトセットアップ](./01-project-setup.md)
**優先度**: 高
**概要**: MCPサーバーの基本構成と環境設定
**依存**: なし
**完了日**: 2025-12-26

### ✅ [チケット02: Google Sheets API連携](./02-google-sheets-integration.md)
**優先度**: 高
**概要**: Googleスプレッドシート「2025年店舗管理シート」への読み書き機能
**依存**: チケット01
**完了日**: 2025-12-26

### ✅ [チケット03: LINE Messaging API連携](./03-line-api-integration.md)
**優先度**: 高
**概要**: LINE Messaging APIから売上報告メッセージを取得（Webhook実装）
**依存**: チケット01
**完了日**: 2025-12-26

### ✅ [チケット04: 売上記録機能](./04-sales-recording.md)
**優先度**: 高
**概要**: 解釈された売上情報をGoogleスプレッドシートに記録
**依存**: チケット02, チケット05
**完了日**: 2025-12-26

### ✅ [チケット05: 税計算ロジック](./05-tax-calculation.md)
**優先度**: 中
**概要**: 税込価格から税抜価格を逆算
**依存**: チケット01
**完了日**: 2025-12-26

### ✅ [チケット06: MCPサーバー実装](./06-mcp-server-implementation.md)
**優先度**: 高
**概要**: FastMCPを使用してMCPサーバーを構築し、全機能を統合
**依存**: チケット02, 03, 04, 05
**完了日**: 2025-12-26

### ⬜ [チケット07: テストとデプロイ](./07-testing-deployment.md)
**優先度**: 高
**概要**: 総合的なテストを実施し、本番環境へデプロイ
**依存**: チケット06

## 推奨作業順序
1. チケット01（プロジェクトセットアップ）
2. チケット05（税計算ロジック）← 他の機能と独立しているため並行可能
3. チケット02（Google Sheets API連携）
4. チケット03（LINE API連携）← チケット02と並行可能
5. チケット04（売上記録機能）
6. チケット06（MCPサーバー実装）
7. チケット07（テストとデプロイ）
