### 計算式
```python
単価（税抜） = 税込価格 / 1.1
```

### 端数処理

- または、スプレッドシートの運用ルールに従う

### 例
- 税込 35,200円 → 税抜 32,000円
- 税込 8,800円 → 税抜 8,000円
- 税込 11,000円 → 税抜 10,000円
