## 仕様詳細
### 取得対象
- 最新のトーク履歴（売上報告メッセージ）
- 曖昧な自然言語形式のメッセージ
  - 例：「12/28 PayPalで月4回プラン 35,200円」
  - 例：「今日 現金 パーソナルトレーニング 8000円」

### 出力形式
- Geminiが解釈できる形式でメッセージを返す
- タイムスタンプ、送信者情報を含める
